Angular Commands

ng new [PROJECT_NAME]: For Creation of a new Project.

Usage

$ ng new my-project

ng --version: Outputs the version of the CLI which is being used in the current project.

Usage

$ ng --version

ng add [MODULE/PACKAGE_NAME]: Adds an NPM package and configures the "app.module.ts" file accordingly for usage.

Usage

$ ng add ng2-charts

ng generate [WHAT_TO_GENERATE] [NAME]: Some of the most commonly used "WHAT_TO_GENERATE" are component | service | module | class. These are just some of the commands which generate the boiler plate code for you. The "NAME" is the name of the generated component or service or class or whatever. The location is also determined where should be the generated "WHAT_TO_GENERATE" be saved.

Usage

$ ng generate component components/my-component

This tells angular to generate a new component named "my-component" and add it to the components/ directory in the "app" folder.

$ ng generate service services/my-service

This tells angular to generate a new service named "my-service" and add it to the services/ directory in the "app" folder.

ng serve: Starts the live server on port 4200.

Usage

$ ng serve
